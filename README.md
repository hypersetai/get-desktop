# get-desktop

Distribution repository for **Hyperset Desktop** installers and auto-update feeds.

Source code lives in the private Hyperset monorepo — this repo ships release assets only.

Latest version: **v0.2.3**

## Download

Install from **[hypersetai.com/download](https://hypersetai.com/download)** or the [latest GitHub release](https://github.com/hypersetai/get-desktop/releases/latest).
