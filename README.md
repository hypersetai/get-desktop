# get-desktop

Distribution repository for **Hyperset Desktop** installers and auto-update feeds.

Source code lives in the private Hyperset monorepo — this repo ships release assets only.

Latest version: **v0.2.0**

## Download

Install from **[hypersetai.com/download](https://hypersetai.com/download)** or direct links:

| Platform | Artifact | URL |
| :-- | :-- | :-- |
| macOS (Apple Silicon + Intel) | Universal `.dmg` | [Hyperset.dmg](https://downloads.hypersetai.com/desktop/Hyperset.dmg) |
| macOS (Apple Silicon + Intel) | Universal `.zip` | [Hyperset.zip](https://downloads.hypersetai.com/desktop/Hyperset.zip) |
| Windows x64 | NSIS installer | [Hyperset-Setup.exe](https://downloads.hypersetai.com/desktop/Hyperset-Setup.exe) |

[All releases](https://github.com/hypersetai/get-desktop/releases) · [GitHub assets for v0.2.0](https://github.com/hypersetai/get-desktop/releases/download/v0.2.0/Hyperset.dmg)
